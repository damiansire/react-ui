# react-ui

Reusable React UI components, organized as an npm-workspaces monorepo. Each package is published to npm under its own name, but they evolve together here.

## Packages

| Package | npm | What it does |
|---|---|---|
| [`dynamic-tables`](packages/dynamic-tables) | [npm](https://www.npmjs.com/package/react-dinamic-tables) | Dynamic, interactive tables with cell selection and editing. |
